# Reset Era PRD

## Problem
Overloaded professionals have too many tasks in their head. Traditional to-do apps collect more, not less. The result: decision paralysis, stale lists, and zero execution.

## Solution
Reset Era takes a raw brain dump, extracts tasks, scores them, selects the Brutal 3 (highest-value actions), and freezes everything else. The user gets clarity in under 60 seconds.

## Core loop
1. Dump (unstructured input)
2. Extract + score (AI)
3. Brutal 3 (max 3 tasks with first steps)
4. Execute (one-task focus mode)
5. Reflect (end of day)
6. Deep Freeze (stale task cleanup)
